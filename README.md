## 自傳（約1500字）

　　我叫陳柏銘，目前就讀於弘光科技大學智慧科技應用系二年級。一路走來，我始終相信「技術是工具，思考才是核心」。從國中開始接觸資訊與機械相關課程後，我逐漸對科技如何改善生活產生濃厚興趣，也確立了未來希望朝向資訊與智慧應用領域發展的方向。

　　國小階段，我對事物充滿好奇，喜歡拆解與觀察各種生活中的物品，培養了基礎的邏輯思考與動手能力。進入國中後，我開始接觸較完整的自然科學與技術課程，尤其在數學與生活科技方面表現穩定，也第一次意識到「解題」本身是一件有成就感的事情。

　　高中（高職）就讀僑泰高級中學機械科，這段時間讓我真正建立起工程思維。透過機械製圖、加工實習與專題製作，我學會如何將抽象概念轉化為具體成果，也培養了耐心與責任感。雖然機械背景偏向硬體，但我逐漸對程式設計與自動化產生更大的興趣，這也成為我選擇進入智慧科技應用系的重要原因。

　　進入大學後，我開始系統性地學習程式設計、資料處理與智慧應用相關知識，包含基礎程式語言、資料分析、感測器應用與系統整合。相較於大一時期的摸索，大二開始我更能掌握學習節奏，成績與實作能力皆有明顯進步。我也嘗試將課堂所學應用在實際專題與個人練習中，例如製作簡單的網頁、自我介紹頁面與資料整理工具。

　　在課外時間，我積極參與社團與活動，累積團隊合作與溝通經驗。我認為工程與科技工作並非單打獨鬥，而是需要清楚表達想法、理解他人需求，才能完成真正有價值的作品。透過不同角色的嘗試，我學會在團隊中配合、承擔與反思。

　　未來，我希望能持續精進資訊與智慧科技相關能力，結合過往機械背景，朝向跨領域整合發展。不論是在系統應用、資料分析或軟硬體整合方面，我都期許自己成為一位能解決問題、也能持續學習的工程人才。
